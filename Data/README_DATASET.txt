# Dataset Snapshot

This repository only contains a lightweight snapshot of the original IPL ball-by-ball dataset.

Reason:
The complete dataset used for model training is larger than 100 MB and therefore exceeds normal GitHub upload limits.

The full dataset was used locally for:
- feature engineering,
- model training,
- calibration,
- and artefact generation.

The file included here (`IPL_snapshot.csv`) preserves the original column structure and provides a small representative sample for inspection, reproducibility checks, and pipeline validation.
